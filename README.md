# Angular 
## ローカルサーバ起動
```
ng serve
```

## コンポーネント作成
```
ng generate component {component name}

# 例
ng generate component Home
```

## インターフェース作成
```
ng generate interface {interface name}

# 例
ng generate interface housinglocation
```

## サービス作成
```
ng generate service {service name}

# 例
ng generate service housing
```
